#  Turkish Movie Sentiment Analysis AI

##  Project Overview
This project is an AI-powered tool that analyzes Turkish movie reviews to determine whether the sentiment is **Positive** or **Negative**. It uses **Natural Language Processing (NLP)** and Machine Learning to classify comments with high accuracy.

##  Features
- **Scraping:** Automatically collects movie reviews from websites using `curl_cffi` and `BeautifulSoup`.
- **Preprocessing:** Cleans Turkish text (removes stopwords, punctuation, lemmatization).
- **Machine Learning:** Uses **Logistic Regression** with **TF-IDF Vectorization**.
- **Interactive UI:** Built with **Gradio** to provide a user-friendly interface for testing.

##  Results
- **Accuracy:** 89% on the test set.
- **Precision/Recall:** ~0.89 across both classes.

##  Technologies Used
- **Language:** Python
- **Libraries:** Pandas, Scikit-learn, NLTK, BeautifulSoup, Gradio.
- **Environment:** Google Colab.
